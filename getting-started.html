Getting Started

As a developer writing an integration with Egnyte, these are the first steps you take:

  1. Browse this cook book briefly to get the idea for what you can use
  2. Get in touch with your partner account manager to get access to a developer environment. Let them know if you'll use UI Integration Framework of File Picker Service
  3. Set up an account on API Portal
  4. Check if your API key works with your developer environment
  5. Choose the recipes you want to follow and... happy coding!

Your API key you got from the registration will initially be limited to only access one Egnyte domain. That domain will probably be your developer environment. Your key limitation will be lifted once you get certified.

While you're exploring our APIs you can also try our API examples connection in Postman

If you're working with the JavaScript SDK, see Getting JavaScript SDK into your front-end pipeline
